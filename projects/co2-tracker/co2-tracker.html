CO2‑Tracker

Eine interaktive, responsive Web‑App, die CO₂‑Emissionsdaten aus Tabellenquellen zugänglich macht – mit Filterung, Diagrammen und Fokus auf Barrierefreiheit & Inklusion.

Stack: HTML · CSS (Bootstrap/Tailwind‑kompatibel) · JavaScript · XLSX‑Parser · Chart.js

GitHub‑Repository Live‑Demo öffnen Zur Case Study

1. Hintergrund & Motivation

Ziel ist eine öffentlich zugängliche Website, die Transparenz schafft: Welche Unternehmen und Länder emittieren wie viel CO₂ pro Jahr? Forschung zeigt, dass verständliche Daten & klare Visualisierungen Verhalten beeinflussen und die Relevanz von Klimaschutz erfahrbar machen. [M1][M2][M3][M4]

Informationsarchitektur mit klarer Startseite (Intro, Filter, Tabelle, Diagramm) und separater graph.html für Visualisierungen. 4. Prototyp & Ergebnisse

Excel → JSON: mittels XLSX‑Bibliothek in einer async‑Funktion, die fetch & ArrayBuffer nutzt. Anschließend Mapping auf ein sauberes Objektmodell und Rendering in der HTML‑Tabelle via displayData(). 

async function fetchData(){
  const res = await fetch('assets/data.xlsx');
  const buf = await res.arrayBuffer();
  const wb = XLSX.read(new Uint8Array(buf), { type: 'array' });
  const ws = wb.Sheets[wb.SheetNames[0]];
  const rows = XLSX.utils.sheet_to_json(ws);
  return rows.map(r => ({ country:r.country, company:r.company,
    date:r.date, sector:r.sector, mt:r['MtCO2 per day'] }));
}

Filter: einfache applyFilters()‑Pipeline + change‑Listener. Charts: separate graph.html nutzt Chart.js für Liniendiagramme.
